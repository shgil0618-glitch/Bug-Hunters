## 📂 포트폴리오

### 1. 주제 선정

 * **메뉴 추천 플랫폼**

### 2. 필요 기능 및 홈페이지 레이아웃 구상
```
- 1. 회원가입(로컬&api) / 로그인 / 회원관리 / 사용자권한 분리
- 2. 메뉴&레시피 게시판 글 작성(ai기반 자동입력) / 수정 / 불러오기 / 비속어 검출api
- 3. 레시피 재료 상세 정보 표기
- 4. 레시피 재료 판매(쇼핑몰 api연결)
- 5. 메뉴&레시피 게시글 댓글/좋아요/댓글 총평(ai기반 댓글 한줄평)
- 6. 댓글 자동번역api
- 7. 게시글 페이징 처리 및 광고배너 삽입
- 8. 구독시스템 도입(구독 등급에 따른 마일리지 제공&구독자 전용 컨텐츠 ex.쉐프 비밀레시피)
- 9. 마일리지로 사이트내 상품구매가능(현금과 재화가치는 같음)  
```
